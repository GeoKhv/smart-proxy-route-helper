# Russian Detailed Description

```text
Smart Proxy Route Helper направляет выбранные сайты через уже запущенный на вашем компьютере локальный прокси, а остальные сайты по умолчанию открывает напрямую.

Как это работает

Установите и запустите V2Ray, V2RayN, V2RayU или другой совместимый прокси-клиент, затем укажите в расширении локальный адрес и порт. Для нужных доменов создайте правила Proxy, а для исключений — правила Direct. Расширение формирует конфигурацию маршрутизации локально и применяет её средствами Chrome.

Основные возможности

• правила Proxy и Direct для отдельных доменов;
• точные правила для одного hostname и правила с включением поддоменов;
• понятный статус текущего сайта и быстрые действия во всплывающем окне;
• редактирование правил с предварительным просмотром изменения области действия;
• экспорт и импорт резервной копии настроек с предпросмотром перед применением;
• приведение стандартного адреса вида www.example.com к example.com при создании новых целей маршрутизации.

Поиск дополнительных доменов

Современный сайт может загружать изображения, файлы и другие ресурсы с дополнительных доменов. По явной команде Smart Proxy Route Helper ищет такие hostname на текущей странице и показывает кандидатов для проверки. Каждый кандидат можно добавить отдельно, а выбранные кандидаты — одной общей командой.

Если дополнительный домен появляется только во время конкретного действия — например, при загрузке или скачивании файла, воспроизведении медиа либо открытии раздела страницы, — можно заранее включить временную запись, выполнить действие, затем остановить запись и просмотреть найденные hostname. Ни предварительный поиск, ни запись не создают правил автоматически: добавление всегда требует подтверждения пользователя.

Синхронизация и локальные настройки

Доменные правила и настройки классификации могут синхронизироваться между установками Chrome через Chrome Sync. Адрес, порт, протокол и состояние локального прокси хранятся только на текущем устройстве и не синхронизируются расширением. Резервную копию можно экспортировать и импортировать локально; данные локального прокси по умолчанию в неё не входят.

Конфиденциальность и local-first модель

У расширения нет backend, телеметрии, аналитики, рекламы, удалённого списка правил или удалённого исполняемого кода. Логика маршрутизации и классификации поставляется вместе с расширением и выполняется локально.

Через временный мост поиска связанных доменов передаются только hostname. Расширение не сохраняет и не передаёт пути URL, query-параметры, заголовки, cookies, учётные данные, содержимое страниц или файлов. Найденные hostname используются локально для предпросмотра и не становятся правилами без отдельного подтверждения.

Что требуется от пользователя

Smart Proxy Route Helper не предоставляет VPN, прокси-сервер или собственную сеть. Для маршрута Proxy необходим уже работающий локальный прокси-клиент, который вы устанавливаете, настраиваете и контролируете самостоятельно. Если правило не совпало, сайт открывается напрямую по маршруту DIRECT.
```
