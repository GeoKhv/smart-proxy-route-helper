# Russian Privacy Summary

```text
Smart Proxy Route Helper работает по local-first модели: у расширения нет backend и телеметрии. Оно не сохраняет полные URL, пути и query-параметры и не собирает содержимое страниц или файлов.

Адрес и порт локального прокси, его состояние и настройка языка интерфейса «Авто (Chrome)» / English / Русский остаются на текущем устройстве. Настройка языка — обычная локальная настройка интерфейса, а не данные, собираемые разработчиком. Доменные правила маршрутизации и настройки классификации могут синхронизироваться через Chrome Sync. Временный поиск связанных доменов обрабатывает только hostname, а добавление каждого правила требует подтверждения пользователя.
```

This reference does not change Chrome Web Store Privacy practices. The current conservative Dashboard categories remain:

- Browsing history.
- User activity.
- Website content.
