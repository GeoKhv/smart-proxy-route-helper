# Russian Permission Justifications

These translations are reference text for Store review and support. This slice does not change the existing Chrome Web Store Dashboard declarations.

## `proxy`

```text
Разрешение необходимо, чтобы применять в Chrome локально сформированную PAC-конфигурацию. Расширение использует её для правил Proxy и Direct, созданных пользователем, и направляет выбранные домены через указанный пользователем локальный прокси. Удалённые PAC-файлы и удалённая логика маршрутизации не используются.
```

## `storage`

```text
Разрешение необходимо для хранения настроек расширения. Доменные правила маршрутизации и настройки классификации могут храниться в Chrome Sync. Адрес, порт, протокол и состояние локального прокси хранятся только на текущем устройстве. Краткоживущие метаданные активной записи могут храниться в session storage; записанные hostname там не сохраняются.
```

## `activeTab`

```text
Разрешение используется только после явного действия пользователя в текущей вкладке. Оно поддерживает управление маршрутом текущего сайта, ручную проверку через прокси, поиск связанных доменов и запуск, остановку или отмену временной записи для конкретного действия. Расширение не запрашивает постоянный доступ ко всем сайтам.
```

## `scripting`

```text
Разрешение необходимо для запуска временных скриптов, поставляемых вместе с расширением, на активной странице после явного действия пользователя. Эти скрипты используются для поиска hostname ресурсов текущей страницы и для ограниченной по времени записи hostname, появляющихся во время конкретного действия. Расширение не объявляет постоянные content scripts и не использует удалённый исполняемый код.
```

## Manifest cross-check

The current manifest permissions are exactly:

```text
proxy
storage
activeTab
scripting
```

The manifest has no `host_permissions`, `<all_urls>`, `webRequest`, `webNavigation`, or persistent content scripts.
