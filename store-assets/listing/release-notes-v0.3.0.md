# Smart Proxy Route Helper v0.3.0 Candidate Release Notes

Prepared for a future release. The manifest and package versions remain `0.2.0`, and these notes do not state that v0.3.0 has been published.

## English

```text
Candidate notes for v0.3.0:

- Made related-domain confirmation clearer with an Add action beside every saveable candidate.
- Added a sticky batch action for selected related domains.
- Added English and Russian interface localization.
- Canonicalized standard WWW hostnames, so addresses such as www.example.com use the example.com route target.

Related-domain preview and recording remain user-invoked, hostname-only, and unable to create rules without explicit confirmation.
```

## Русский

```text
Предварительные заметки для v0.3.0:

- Добавление связанных доменов стало понятнее: у каждого доступного кандидата появилась отдельная кнопка «Добавить».
- Для выбранных связанных доменов добавлено закреплённое групповое действие.
- Добавлена локализация интерфейса на английский и русский языки.
- Стандартные WWW-адреса канонизируются: например, www.example.com использует цель маршрутизации example.com.

Поиск и запись связанных доменов по-прежнему запускаются только пользователем, обрабатывают только hostname и не создают правила без явного подтверждения.
```
