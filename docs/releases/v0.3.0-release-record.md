## Known limitations

- A compatible user-configured local proxy is still required.
- Chrome proxy settings may be controlled by enterprise policy, another extension, or user settings.
- Related-domain preview and recording remain bounded page-level signals and may not observe worker,
  service-worker, extension, or browser-level requests.
- Proxy authentication management is not included.
- The low-severity `esbuild` development-server advisory remains open for a separate dependency
  maintenance decision.
- Post-release fixes on `main` are not part of the immutable `v0.3.0` tag or GitHub asset unless a
  future release is created explicitly.
