# English Privacy Summary

```text
Smart Proxy Route Helper follows a local-first model: it has no backend, telemetry, analytics, or advertising. It does not store raw URLs, URL paths, query parameters, page contents, or file contents.

The local proxy address, port, protocol, and enabled state remain on the current device. Domain route rules and classification overrides may sync through Chrome Sync. Related-domain preview and action-specific recording use a hostname-only bridge and are temporary. A route rule is created only after the user explicitly selects and confirms it.
```
